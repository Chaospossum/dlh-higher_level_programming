# Python - import & modules

This project covers Python imports, modules, and command-line arguments.

## Tasks

- 0-add.py: Imports the add function from add_0.py
- 1-calculation.py: Imports calculator functions from calculator_1.py
- 2-args.py: Prints command-line arguments
- 3-infinite_add.py: Adds all command-line arguments
- 4-hidden_discovery.py: Lists names from a compiled module
- 5-variable_load.py: Imports a variable from another file
